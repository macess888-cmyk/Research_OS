# Research OS

A Streamlit-based academic portfolio and research dossier system for managing biography, publications, software, teaching, collaborations, timelines, and research outputs from Markdown source files.

## Current Instance

Jake Macdonald Academic Portfolio

## Features

- Modular Streamlit app
- Markdown content source
- Sidebar navigation
- Research dashboard
- Research Atlas
- Admin Editor
- Build Center
- Backup export
- Live project statistics
